# Suffix Array Guide

## Overview
Suffix Array is a sorted array of all suffixes of a string, used for efficient pattern matching.

## Time Complexity
- Build: O(n log n) or O(n) with advanced algorithms
- Pattern search: O(m log n)
- Space: O(n)

## Use Cases
- Pattern matching in text
- Longest common substring
- Bioinformatics (DNA sequence analysis)
- Data compression

## Summary
Suffix arrays provide efficient string matching with less space than suffix trees.
